## 🔧 Technical Details

### Technologies Used
- **Framework**: Next.js 14 with App Router
- **Styling**: Tailwind CSS 3.3
- **Animations**: Framer Motion
- **Icons**: Heroicons
- **TypeScript**: Full type safety
- **Build System**: Turbopack

### File Structure
